# Vant H5 Template

## Environment

```shell
# use specified version of node https://github.com/nvm-sh/nvm
nvm use
# install dependencies
pnpm i
# dev
pnpm run serve
# build
pnpm run build
# lint
pnpm run lint
# deploy
pnpm run deploy:test
pnpm run deploy:production
# chmod +x
chmod +x ./scripts/deploy.sh
```
